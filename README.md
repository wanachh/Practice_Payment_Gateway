# Practice_Payment_Gateway

My mock project to understand about payment gateway and the structure
mostly part of this project I use Gemini to guide and explain code


I have 2 solutions in this project

1.ZenPay.Core for logic, models, and data definition.

    - Data/ZenPayDbContext.cs: database bridge.

    - Migrations/: The InitialCreate migration.

    - Models/Transaction.cs: Transaction data definition.

2.ZenPay.Api for outside world requests.
